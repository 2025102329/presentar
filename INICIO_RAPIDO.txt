DE LA CRUZ FRAGANCIAS IA - INICIO RAPIDO
========================================

1. Abre esta carpeta completa en Visual Studio Code.
2. Abre el archivo .env.
3. Reemplaza PEGA_AQUI_TU_API_KEY por una API key NUEVA.
4. Guarda el archivo.
5. Ejecuta run-windows.bat.

Tambien puedes usar la terminal:

mvn spring-boot:run

Luego abre:
http://localhost:8081

PRUEBA DE IA
- Registra dos clientes.
- Abre Arbol IA.
- Entra en Decisiones IA.
- Si aparece GEMINI, la API respondio correctamente.
- Si aparece LOCAL, se utilizo el respaldo del sistema.
